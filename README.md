# Para executar o projeto

1️⃣ - Clone o repositório 

2️⃣-  Abra a pasta do projeto

3️⃣ - Instale as dependências com comando: 

```sh
npm install 
```
4️⃣ - Para iniciar o app execute o comando:

```sh
npx expo start
```
